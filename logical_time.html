Scalar Time

The scalar time representation was proposed by Lamport in 1978 as an attempt to totally order events in a distributed system.

Vector time

The system of vector clocks was developed independently by Fidge [4], Mattern [12], and Schmuck [23]. In the system of vector clocks, the time domain is represented by a set of n-dimensional non-negative integer vectors.
